# Solo Project - Unit converter
- This app allows users to convert a value from one unit to  the other
- Built with HTML,CSS & JS

## Features
- Allows users to input value
- Convert button generate conversions when clicked

## Getting Started

Install the dependencies and run the project

```
npm install
npm start
```

## Deployment

- Deployed on netlify
- livesite URL: (https://unitconverrt.netlify.app/)
